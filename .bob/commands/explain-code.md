# /explain-code - Plain English Code Explainer

## Description
Translates complex code into plain English explanations that anyone can understand. Perfect for onboarding new developers, documenting legacy code, or understanding unfamiliar codebases.

## How It Works
When you run `/explain-code [file-path]` or `/explain-code [file-path:line-range]`, Bob will:
1. Read the specified code
2. Analyze the logic and flow
3. Explain what the code does in simple terms
4. Break down complex algorithms step-by-step
5. Identify potential issues or edge cases
6. Suggest improvements if applicable
7. Auto-save the explanation to `output/code-explanation.md`

## Usage Examples

```bash
# Explain entire file
/explain-code src/utils/auth.js

# Explain specific lines
/explain-code src/utils/auth.js:45-120

# Explain function by name
/explain-code src/utils/auth.js:validateToken
```
